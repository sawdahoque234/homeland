# [Homeland](https://homelandproject-bb634.web.app/)

# Project Features

## User Context

- Login using Email/Google
- My Profile
- Buy Property
- Order Review
- Order Cancel
- Payment using Card
- Check Status of Order (pending, confirm etc)
- Adding a review

## Admin Context

- My Profile
- Manage User
- Manage Properties
- Manage Orders
- Add Properties
- Add Admin

# Tools & Technology Used (Client)

- React.js
- Firebase Auth
- Stripe (Card Payment)
- React router dom
- React-hook-form
- AOS-Animation
- React Icons
- TailwindCSS
- DaisyUI
- Git Version Control + GitHub
- Firebase Hosting (Client Deploy)

# Tools & Technology Used (Server)

- Express.js
- MongoDB
- Heroku (Server Deploy)
